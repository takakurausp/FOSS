Test Mode Implementation Demo

テストモードの動作説明

テストモードの有効化方法:

URLに ?test=true パラメータを追加してアクセスします。

例: https://script.google.com/.../exec?test=true

テストモードの動作:

  • 新規投稿画面が表示されます（通常は制限されています）
  • フォーム項目があらかじめ入力されています
  • 責任著者のメールアドレスは空欄です
  • 共著者のメールアドレスは example.com ドメインを使用

コード実装箇所:

  1. Code.js (行42): template.isTestMode = (e.parameter.test === 'true') ? 'true' : 'false';
  2. index.html (行30): const IS_TEST_MODE = "<?= isTestMode ?>" === "true";
  3. scripts.html (行549-557): テストモードチェックと制限メッセージ
  4. scripts.html (行560-574): テストデータ定義
  5. scripts.html (行579-769): テストモード用フォーム表示

テストモードの確認方法:

実際のGoogle Apps Scriptアプリに ?test=true パラメータを付けてアクセスし、以下の点を確認:

  1. 新規投稿画面が表示される（通常モードでは制限メッセージ）
  2. フォーム項目にテストデータが入力されている
  3. 責任著者メールアドレスが空欄
  4. 共著者メールアドレスが example.com ドメイン
