テストモードデバッグ

デバッグ情報

このページはテストモードの動作を確認するためのものです。

ブラウザの開発者ツール（F12）のコンソールを開いて、ログを確認してください。

テストリンク

1. 通常モード（新規投稿不可）

通常モードで開く

URL: /dev

2. テストモード（?test=true）

テストモードで開く

URL: /dev?test=true

3. テストモード明示（?test=true）

テストモード（key付き）で開く

URL: /dev?test=true&key=test123

4. テストモード別パラメータ

?test=1 で開く

URL: /dev?test=1

5. テストモード大文字

?test=TRUE で開く

URL: /dev?test=TRUE

期待される動作

  • リンク1: 「新規投稿はできません」と表示される
  • リンク2: テストモードの新規投稿フォームが表示される
  • リンク3-5: テストモード判定ロジックの動作を確認

コンソールで確認すべきこと

  1. 「All URL parameters:」のログでパラメータが正しく取得されているか
  2. 「test parameter:」のログでtestパラメータの値
  3. 「isTestMode template variable:」のログでテンプレート変数の値
  4. 「IS_TEST_MODE value:」のログでフロントエンドでの値
